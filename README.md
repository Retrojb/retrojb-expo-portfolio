## Usage

Running the project locally
- `$ yarn `
- `$ yarn start ios || android || web`

Clearing Mobile caches:
- ` $ yarn clear-all` - Removes node_modules, watchman, metro, derived data
- ` $ yarn clear-caches` - Removes watchman, metro, derived data
